## MacOS Speed up
* **Change `<Caps-Lock>` to `<Esc>`** in MacOS keyboard settings (Modifier Keys...)

* **Change your keyboard settings**
  - Set **Key Repeat** `(- - - - - - - *)`
  - Set **Delay Until Repeat** `(- - - - - - - *)`
